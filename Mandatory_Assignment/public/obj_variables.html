Primitive JavaScript Datatypes & Structure Types

JavaScript is a loosely typed and dynamic language. Variables in JavaScript are not directly associated with any particluar value type, and any variable can be assigned and re-assigned as values of all types.

let numberOne = 1;

numberOne = "Number One";

Primitive Datatypes
A primitive datatype is data that is not an object and has no methods. All primitives are immutable and cannot be altered.
In example:

let bar = "baz";
bar.toUpperCase();
console.log(bar); // will return baz, because it is immutable and can not be altered.
bar = bar.toUpperCase();
console.log(bar); // will return BAZ, because it is assigned a new value so it can be replaced but not altered.

There are seven data types in JavaScript, that are considered primitive.
  • Undefined
  • Boolean
  • Number
  • String
  • BigInt
  • Symbol
  • Null

They can be declared with different declarators & they are all hoisted to the top of their scope, meaning they can be used before they are declared

var numberOne = 1;

  • var declarations are globally scope or function scoped, their variables can be updated and re-declared within its scope. var can be declared without being initialized.

let numberTwo = "2";

  • let is block scoped, let variables can be updated but not re-declared. let can be declared without being initialized.

const numberThree = "Three";

  • const is block scoped, ocnst variables can neither be updated nor re-declared, const must be initialized during declaration.

Then to the structural types
All the structural types are derived from Object, therfore they are all objects and have methods assigned to them, which we can use to manipulate the datastructures as we need. Almost everything made with the new keyword is a data structure.
Underneath i've listed some.
Structural types
  • Object
    • Objects can be seen as a collection of properties, which can be added or removed. Properties can be any type. It is also seen as a mapping between keys and values.

      const person = {firstName:"John", lastName:"Doe", age:50, eyeColor:"blue", ugly: true}; console.log(person.firstName); // output = John

  • Array
    • Arrays are data-structures that comes with a set of methods to manipulate the data it can contain on each index it is perfect to represent a list of something. It can hold a variety of data types. some of the methods are:


      indexOf(); // which will take a parameter and look through the array for a match and return the index of its first occurance. push(); & pop(); // Are used to add or remove data, at the end of an array.
      const months = ['Jan', 'March', 'April', 'June'];
      months.splice(1, 0, 'Feb'); // inserts at index 1 if the 0 was a 1 it would remove 1 from the position given and add the "Feb".
      console.log(months[1]); // will print "March"

  • Date
    • Date is an data-structure to which you can manipulate with a set of methods. Options to set and get various parts of time and dates.


      const today = new Date(); // creates a new Date object.
      const dayDate = (today.toLocaleString('en-DK', { day: 'numeric' })); // declares a variable that contains the day of the month in number format.
      const month = (today.toLocaleString('en-DK', { month: 'long' })); // month in string format
      const time = (today.toLocaleTimeString('en-DK')); // time in 00:00:00 format
      const day = (today.toLocaleString('en-DK', { weekday: 'long' })); // day as Friday in string
