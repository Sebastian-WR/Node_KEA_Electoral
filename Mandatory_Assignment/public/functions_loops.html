It is executed when something invokes it (calls it).

Callback Functions
A callback is a function that we pass as an argument in another function.
Follow the example below.

function sayHiLater(anyFunctionReference) {
// simulate some code running
// ... 5 seconds later ....
anyFunctionReference();
}

function hiFunction () {
console.log("Hi!");
}

sayHiLater(hiFunction); // returns "Hi!"

Here you can se that the callback is the function in the parameter so the method that takes the function can run code and then use the other function later on.
that is what's called a callback function.
Loops
Loops offer a quick and easy way to do something repeatedly. It is a way of iteration through data structures like arrays, lists, et cetera.
Traditional loops:
  • For loop --- Loops a fixed number of times
  • For each loop --- Loops through th whole thing
  • While loop --- Loops as long as condition is true
  • Do while loop --- Loops as long as condition is true
Here are the syntax for the listed loops in the same order:

for (let index = 0; index < array.length; index++) {
// ... some code ...
}

array.forEach(element => {
// ... some code ...
});

while (condition) {
// ... some code ...
}

do {
// ... some code ...
} while (condition);

Alternative Loops
  • Map --- Returns an array with the same lenght
  • Filter --- Returns an array with less after filtering
  • Find --- Returns the first item in an array thatsatisfies a condition
Here are the syntax for the listed loops in the same order:

const failures = ["misclick", "falling", "tiktok", "Christopher"];

const failuresCopy1 = failures.map((failure, index) => console.log(failure, index));

const failuresCopy2 = failures.find((failure) => failure === 'tiktok');

const failuresCopy3 = failures.filter((failure) => failure === 'tiktok').shift();
