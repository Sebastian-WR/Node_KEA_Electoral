Terminal Commands

Goin through various commands i have used in this class, and their behaviour.

Git commands
  • git add [file] --- Will add the given file to the commit.
  • git add -A --- Will add all files to the commit.
  • git commit -m "[commit message]" --- Prepares the commit and takes a commit messages to describe the commit.
  • git push --- Will push the commit to remote repository.
  • git pull --- Will pull all changes from remote repository.
  • git init [repository name] --- Will create a new repository.
  • git clone [url] --- Will clone an existing reopsitory from its URL.
NPM commands
  • npm install --- Will install node modules
  • npm install [package name] --- Will install the given package locally.
  • npm install -g [package name] --- Will install package globally.
  • npm --version --- Will lstate your current version.
  • npm ls --- Will list the current global packages installed.
  • npm uninstall [package name] --- Will uninstall the given package.
  • npm install [package name]@[version number] --- Will install specific version of given package.
  • npm outdated --- Will list outdated packages.
  • npm update [package name] --- Will update given package.
  • npm run [script name] --- Will run a script from package.json example below.

"scripts": {
"start": "node app.js",
"start-dev": "cross-env PORT=3000 nodemon.cmd app.js"
}

The scripts are given a name first like start & start-dev. The commands they should run are given afterwards.
start-dev will set the environment port to 3000 and start the server with nodemon.
cross-env makes sure that the setting of the port works on all OS.
Terminal & Node commands
  • node [API file name] --- Will run your web server.
  • nodemon.cmd [API file name] --- Will run your web server & restart it everytime changes have been made.
  • CTR + C --- Will exit your running nodemon if pressed twice.
  • cd .. --- Will go back one directory in the terminal.
  • cd [folder in directory] --- Will take you to the given folder, you can tab after the first letter to autocomplete.
  • ls --- Will list fiels and diretories in the current directory.
  • mkdir [directory name] --- Will make a new directory with given name.
  • rmdir [dir name or path] --- Will remove directory
  • clear --- Will clear the terminal
  • help --- Will list help in terminal
  • man [command] --- Will list info on the given command
